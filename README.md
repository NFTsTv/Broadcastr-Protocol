# Broadcastr Protocol

Contracts for the Broadcastr protocol.

## For contributing
`git clone https://github.com/NFTsTv/Broadcastr-Protocol`

`cd Broadcastr-Protocol`

`git checkout -b yourbranchname`

`forge install`
